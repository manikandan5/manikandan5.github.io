Projects

  Weather Predictor - Developed a fault tolerant prototype which is built on Service Oriented Architecture with multiple micro services communicating with each other through REST calls. Deployed the services on Amazon EC2 instances with the database hosted on RDS. The prototype features continuous integration, continuous deployment and load balancing.

  Stereogram The program creates a stereo image from the normal image and the disparity map from the same view. If a seed image is given instead of a disparity map is not given, it tries to estimate the disparity using Markov Random Field energy minimization.

  Image Matching, Stitching and Warping Given two images of the same object, but of different view, the program tries to find the matching SIFT points. Then it estimates the homography and performs the projective transformation so that the second image is transformed into the same view as the first image.

  Optical Music Recognition Program to find the music notes from the picture of a music sheet. It uses image recognition techniques to identify the same.

  Object detection The project classifies the objects in every image into a pre-defined class. In this project, we have compared the results found using a simple baseline(subsampled pixel points passed to a SVM), Principal Component Analysis (PCA), Haar-like features (Viola-Jones), Bag Of Words model and a Convolutional Neural Network.

  Scene Classifier Scene classification is a problem under the domain of object recognition. Scene classification is a very interesting problem of automatically labeling an image among a set of semantic categories. To achieve this, we had implemented a Bag Of Words model and used Cornell's SVMLight to train and classify the data.

  YELP Dataset Challenge My attempt at the Yelp Dataset Challenge that could be found in the YELP site . Predicted the category of the restaurant by training the classifier on the given dataset.

  Search-Indexation Indexation of documents provided using Lucene Package.

  Search-Retrieval Implementation of various retrieval algorithms using Lucene Package.

  Search-QueryCentricPageRank Implementation of Query Centric Page Rank algorithm using Lucene and JUNG packages.

  Shopping Website A shopping website with which lets the user register, view and order the products on the site and an admin panel to monitor the orders and manage the inventory of the site.

  Image Orientation The program tries to find the right orientation of the given image. It trains on the given dataset and classifies the orientation using K-nearest neighbors and Neural networks.

  POS Tagger Program which classifies the given sentence into its respective part of speech using Bayesian network models and Natural Language Processing techniques.

  Zacate Zacate is traditional dice game played in Latin America. It is a game of chance. The rules of the game can be found in the github repository. The code here is an AI program which takes certain decisions to maximize the score.

  Map Coloring The program address the typical map coloring problem and tries to come up with a solution so that no adjacent states have the same color.

  Heuristic Search and Games AI Player to play Rameses and Totogram using Heuristic Search algorithms such as BFS and Iterative Deepening.
